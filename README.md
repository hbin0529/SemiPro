## 기간
> 2022.12.12 ~ 2022.12.21(10일)
<br>

## 개발환경
> * Java 1.8
> * Oracle DB
> * Jsp
> * Javascript(jQuery, ajax)
> * Apache Tomcat9.0
<br>

## 주요기능
> * 커뮤니티 : CRUD 게시판, 페이징 처리, 댓글, 검색
> * 회원가입 : 회원가입 및 로그인
